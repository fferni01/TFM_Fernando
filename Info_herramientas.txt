Hulk, Slowloris, Thor Hammer y Bonesi son todos tipos de ataques informáticos diferentes que se utilizan con diferentes propósitos y técnicas.

Hulk es un tipo de ataque de denegación de servicio (DoS) que se utiliza para sobrecargar y bloquear un sitio web o servidor. Se realiza enviando una gran cantidad de tráfico falso a la dirección del objetivo.

Slowloris es también un tipo de ataque DoS, pero en lugar de enviar tráfico falso, se realiza enviando solicitudes incompletas a un servidor de forma constante. Esto hace que el servidor se quede esperando a que se complete la solicitud, bloqueando así otros servicios.

Thor Hammer es una herramienta de prueba de estrés utilizada para evaluar la capacidad de un sistema para manejar una gran cantidad de tráfico. Aunque se puede utilizar para probar la resiliencia de un sistema, también se puede utilizar con fines malintencionados para sobrecargar y bloquear un sitio web o servidor.

Bonesi es una herramienta de ataque DoS que se utiliza para enviar tráfico falso a una dirección IP específica. El objetivo del ataque es sobrecargar y bloquear el servidor o sitio web del objetivo.

LOIC (Low Orbit Ion Cannon): una herramienta de ataque DoS que se utiliza para enviar una gran cantidad de tráfico a una dirección IP específica.
DDOSIM: un simulador de ataques DDoS que se utiliza para probar la resiliencia de un sistema frente a ataques DDoS.

R-U-Dead-Yet: una herramienta de ataque DoS que se utiliza para enviar solicitudes HTTP a un sitio web de forma continua, bloqueando así el acceso a otros usuarios.

XOIC: una herramienta de ataque DoS que se utiliza para enviar tráfico falso a una dirección IP específica con el objetivo de sobrecargar y bloquear el servidor o sitio web del objetivo.

GoldenEye: una herramienta de ataque DoS que se utiliza para enviar tráfico HTTP falso a una dirección IP específica.
PyDDoS: una herramienta de ataque DDoS escrita en Python que se puede utilizar para enviar tráfico falso a una dirección IP específica.
HULK: una herramienta de ataque DoS que se utiliza para enviar una gran cantidad de tráfico a una dirección IP específica con el objetivo de sobrecargar y bloquear el servidor o sitio web del objetivo.

LOIC-derived: una herramienta de ataque DoS que se utiliza para enviar tráfico falso a una dirección IP específica con el objetivo de sobrecargar y bloquear el servidor o sitio web del objetivo.
