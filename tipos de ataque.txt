Ataques de flujo: consisten en enviar un gran número de paquetes de datos a una dirección IP específica de manera que se llene el ancho de banda disponible y no se pueda acceder al servidor.
pyLoris
hulk
thoshammer 

Ataques de fragmentación: consisten en fragmentar los paquetes de datos en trozos más pequeños y enviarlos a una dirección IP específica de manera que el servidor tenga que gastar recursos adicionales para reconstruir los paquetes y procesarlos.



Ataques de protocolo: consisten en enviar solicitudes malformadas o inválidas a un servidor de manera que tenga que gastar recursos adicionales para procesarlas y rechazarlas.

slowloris


Bonesi es una herramienta de prueba de estrés y rendimiento para servidores web. En lugar de ser una herramienta de ataque DDOS, Bonesi se utiliza para evaluar la capacidad de un servidor web para manejar una gran cantidad de tráfico y solicitudes simultáneas de manera que se puedan identificar problemas de rendimiento y tomar medidas para solucionarlos.
